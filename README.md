# debiantor
Torify Debian
Based on Kalitorify https://github.com/brainfucksec/kalitorify

Instructions:

> 1. Run Tor: 
```
service tor start
```
> 2. Set permissions and run script as root: 
```
chmod +x debiantor.sh
sudo ./debiantor.sh --start
```
> 3. Stop script: 
```
sudo ./debiantor.sh --stop
```
